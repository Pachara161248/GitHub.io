Git คือ เครื่องมือที่ของโปรแกรมเมอร์ใช้ในการพัฒนา Software ช่วยเก็บรักษาไฟล์ สามารถติดตามการเปลี่ยนแปลงของไฟล์และยังสามารถกลับไปแก้ไขไฟล์เก่าได้
GitHub คือ เว็บไซต์ที่ให้บริการ Gitนั่นเอง โดย Github จะให้บริการบนออนไลน์แพลตฟอร์ม ทำให้คุณสามารถเข้าถึงข้อมูลผ่านหน้าเว็บไซต์ได้ทุกที่ทุกเวลา ในส่วนของการใช้บริการก็มีให้ใช้ทั้งแบบเสียเงินและไม่เสียเงิน หากต้องการใช้บริการแบบฟรี ๆ Code ในโปรเจกต์ของเราทั้งหมด คนอื่นจะสามารถเข้าถึงได้ แต่ถ้าอยากใช้แบบส่วนตัวทาง GitHub ก็มี Plan รองรับในราคาค่าใช้จ่ายที่ไม่สูงมากเช่นกัน
GitLab คือ เว็บไซต์ที่พัฒนามาจาก GitHub ซึ่งสองเว็บไซต์นี้จะใช้งานคลายๆกัน
